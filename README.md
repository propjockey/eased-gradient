# eased-gradient by PropJockey

```js
  const props = {
    draw: "circle at center",
    stops: { "0%": "#ff0000", "100%": "#0000ff" },
    ease: ["factory.steps", 8, "jump-end"],
    freq: 0.067
  }
  easedGradient(props, returnRawData)
```

See `demo.html` and `test/eased-gradient.js` for example code use.
